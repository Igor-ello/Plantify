# Приложение для ухода за растениями

1. Заполняйте карточки растений (их видов и групп)
2. Добавляйте рстения в избранное и в список желаемого
3. Сохраняйте ваши созданные библиотеки для переноса а другое устройство


## Используемые технологии

### UI / UX

Jetpack Compose - построение пользовательского интерфейса

Material Design 3 - единый стиль интерфейса и компонентов

ColorScheme - поддержка светлой и тёмной темы

State management (State / MutableState / Flow) - управление состояниями UI

### Хранение данных

Room (SQLite) - локальная база данных

DAO - доступ к данным через абстракции

Relations (One-to-Many)- растения и связанные с ними фотографии

### А также

Hilt - внедрение зависимостей

Kotlin Coroutines - асинхронные операции

JSON и kotlinx.serialization - формат и технология резервного копирования и восстановления данных

Coil - загрузка и отображение изображений
